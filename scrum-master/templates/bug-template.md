# Bug Template

## Required in Description

**Steps to Reproduce:**
1. [Clear numbered steps]

**Expected:** [What should happen]
**Actual:** [What actually happens]
**Environment:** [Browser/OS/Version]

## Required Fields
- Priority must be set
- Summary under 100 characters

## Validation Rules
✅ PASS: Description has steps + expected + actual + priority set
⚠️ WARN: No environment info, unassigned >24h, vague summary
❌ FAIL: Missing any required section, summary like "broken" or "fix bug"
